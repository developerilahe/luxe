# LUXE E-Commerce Website

A premium fashion e-commerce website with smooth scroll effects, 3D animations, and full Django backend.

## Features
- Smooth section transitions with half-slide effect
- Dynamic product filtering and search
- Shopping cart functionality
- Newsletter subscription
- Contact form with email collection
- Admin dashboard for product management

## Technologies
- Django 5.2
- SQLite3
- HTML5/CSS3
- JavaScript (GSAP animations)
- Swiper.js
